isset、is_null、empty 的区别

  • isset: 只要变量有值并且值不为 NULL 就为 true, 变量的值为 NULL 也算是有值
  • is_null: is_null = !isset, 只要 isset 为真, is_null 就为假
  • empty: ””/0 /“0”/NULL/FALSE/array()/var $var/没有任何属性的对象, 都会返回 true, 也就是为空

常量的声明与使用

  • 常量是一个简单值的标识符
  • 常量定义后不能再改变他的值, 也不能使用 unset() 取消
  • 常量可以不用理会变量范围的规则而在任何地方都可以定义和访问
  • 常量使用 define("常量名", 值);
  • 常量声明名在声明和使用都不使用 $
  • 常量名称习惯都使用大写, 并用 _ 区分单词
  • 常量的值只能用标量类型(int, float, bool, string)
  • 常量一定要在声明时就给值

  • 比较运算符 : >、<、>=、<=、==、===、!= 或 <>、!==
  • 逻辑运算符 : && 或 and || 或 or ! 或 not
  • 位运算符 : & ｜ ^ ~ << >> >>>

% 有两个作用

  • 整除运算
  • 控制范围, 不要用小数, 也不要用负用, % 会把两边的数转为整数后再进行整除

短路问题

&& 在作运算时, 如果前面的数是 false, 则后面是否为 true, 整个表达式都是 false, 所以就不去执行后面的操作数, 称为短路
|| 在作运算时, 如果前面的数是 true, 则后面是否为 false, 整个表达式都为 true, 所以就不去执行后面的操作数, 称为短路
& 以及 | 运算时, 两边的操作数都会被执行

函数

函数的定义

  1. 函数是一个被命名的
  2. 独立的代码段
  3. 它执行特定任务
  4. 并可以给调用它的程序返回一个值

函数的优点

  1. 提高程序的重用性
  2. 提高程序的可维护性
  3. 可以提高软件的开发效率
  4. 提高软年的可靠性
  5. 控制程序的复杂性

函数的声明

注意

  • 函数必须调用才能执行, 可以在声明之前调用, 也可以在函数声明之后调用
  • 函数名命名和变量一样,  aaa bbb ccc aaaBbbCcc, 函数名称一定要有意义
  • 函数在声明时不能重名
  • 可以通过向函数传递参数, 改变函数的行为
    • 形参: 就在声明函数时, 声明的参数, 参数就是变量, 多个参数用 , 分开
    • 实参: 调用函数时传给形参数值(数据, 也可以变量)
  • 如果没有返回值则称为过程
  • 通过使用 return 语句返回数据
  • 函数执行到 return 语句就结束, 所以不要在这个语句后写代码, 也可以使用 return 结束函数的执行

函数名的作用

  • 调用函数, 开始执行函数
  • 可以向函数中传递 数据
  • 函数名就是返回的值

PHP的变量的范围

局部变量: 在函数中声明的变量就是局部变量, 只能在自己的函数内部使用
全局变量: 在函数外声明, 可以在变量声明以后的, 直到整个脚本结束前都可以使用, 包括在函数中和 {} 中都可使用

注意

  • 参数就是局部变量, 这个局部变量可以在函数调用时去赋值。
  • PHP 的变量分不出 是声明还是使用, 在 PHP 中使用全局变量, 要通过 global 关键字将这个全局变量包含到函数中才能使用到, 在 global 声明之后才是使用全局的变量, 如下

PHP 的静态变量

静态变量只能声明在函数中（类中）, 不能在全局声明, 声明时在变量前使用 static 关键字

作用

变量的值不会随着函数的结束而重置为初始值

注意

  1. 静态变量在静态代码段中保存
  2. 一个函数多次调用之间共用, 但只在第一次调用函数时声明到内存, 以后再调用, 就不再声明, 而直接使用

变量函数

如果一个变量后面有括号 $var=hello; $var();, 就将寻找与变量值同名的函数 hello() 并调用

函数的设计

  1. 函数的功能: 决定是否使用这个函数
  2. 函数的参数: 决定函数怎么调用(有几个参, 传什么类型的值)
  3. 函数的返回值: 调用后怎么处理这个函数

系统函数分类

一定要先使用系统函数, 如果系统函数没有你想要的功能, 再去自己定义函数

  1. 常规函数:  bool copy ( string source, string dest )
  2. 带有mixed (mixed 表示可以传任何类型的数据): bool chown ( string filename, mixed user )
  3. 带有 & 参数的函数(表示引用赋值): bool arsort ( array &array [, int sort_flags] ), 这个参数不能传值, 只能传一个变量, 函数对参数的修改会应用在实参变量上, 也就是说实参的值会随着函数的调用而改变
  4. 带有 [] 的函数(表示这个参数是可选的): bool arsort ( array &array [, int sort_flags] ), 如果你传值了就使用你传的值, 如果没有传值则使用默认值, 直接在声明函数时, 就为参数给初值。注意: 可选值和必须值, 必须从后向前设置
  5. 带有 ... 的参数函数(...表示可以传任意多个参数): int array_unshift ( array &array, mixed var [, mixed ...] )
  6. 带有 callback: array array_filter ( array input [, callback callback] ), 就是调用这个函数时需要我们传一个函数进来(函数名/函数名字串)

常见的几类自定义函数

  1. 内部函数: PHP 可以在函数内部再声明函数, 目的是让函数只能在函数的内部调用, 就是用来帮助外部函数完成一些子功能的
  2. 递归函数: 就是在自己内部调用自己
  3. 重用函数: 使用自己定义的函数库

文件包含命令(也可以作为函数使用)

  • require: 用于静态包含(文件名不含变量), 可能重复包含同个文件
  • include: 用于动态包含(文件名含有变量), 可能重复包含同个文件
  • require_once: 用于静态包含(文件名不含变量), 只会包含一次
  • include_once: 用于动态包含(文件名含有变量), 只会包含一次

数组

数组的概述

  1. 数组的本质: 管理和操作一组变量, 成批处理
  2. 数组是复合类型
  3. 数组中可以存储任意长度的数据, 也可以存储任意类型的数据
  4. 数组就可以完成其它语言数据结构的功能(链表,  队列,  栈,  集合类), 编写 PHP 程序时 30% 的工作都是在操作数组

数组元素的访问

数组中存有多个单元, (单元称为元素), 每个元素(下标［键］和值), 当访问元素时候, 都是通过下标(键)来访问元素, 如: $arr[0]

数组的分类

按维数(下标个数)分

  • 一维数组: 只有一个数组下标, 如: $arr[0]
  • 二维数组: 只有两个数组下标, 如: $arr[0][1]
  • 三维数组: 只有三个数组下标, 如: $arr[0][1][2]
  • 多维数组: 除了一维数组外的数组, 如: $arr[0][1]...[n], 多维数组也被称为存储数组的数组, 就是在数组中存有其它的数组

按照下标的类型分

PHP 中, 下标的类型 只有 int、string 这两种, 故可以划分为以下两类数组:

  • 索引数组: 下标(索引)是顺序整数
  • 关联数组: 下标(索引)是字符串

数组多种声明方式

直接为数组元素赋值

  • 如果索引下标不给出, 就会 从 0 开始顺序索引
  • 如果给出索引下标, 下一个就会从最大的所以开始增 1
  • 如果赋值时, 出现对前面的下标所对应的元素赋值, 则会覆盖该元素原来的值, 重新赋为新值
  • 混合声明时, 索引和关联不互相影响(关联数组的个数不影响索引数组下标的增长)

使用 array() 函数

  • 默认是索引数组
  • 如果需要设置为关联数组或为索引数组指定下标, 使用键=>值, 如: array('key' => 'value');、array(10 => 'value');
  • 多个成员之间使用 , 分割, 如: array('key' => 'value', 10 => 'value');

使用返回值为数组的函数

如: file();

数组的遍历

使用 for 语句循环遍历数组

  • 部分语言只有这种方式, 如: C/C++
  • PHP 中这种方式不是首选
  • 数组必须是索引数组, 而且下标还必须是连续的, 才能这么遍历, 如果索引数组下标不连序, 或者是关联数组, 则不能这么遍历

使用 foreach 语句循环遍历数组

  • 循环次数由数组的元素个数决定
  • 每一次循环都会将数组中的元素赋值给临时变量

while()、list()、each() 组合循环遍历数组

each() 函数

  • 需要一个数组作为参数
  • 返回值也是一个数组
  • 返回的数组是有 0, 1, key, value 四个下标(固定的)
    • 0和key下标是当前参数数组元素的鍵
    • 1和value下标是当前对数数组元素的值
  • 默认当前元素就是第一个元素
  • 每执行一次后就会将数组指针向后移动, 也就是说, 当前元素已经改变, 变成了下一个元素
  • 如果到最后的元素再执行这个函数, 则返回 false

list() 函数

  • list()=array(); 需要将一个数组赋值给这个函数
  • 数组中的元素个数, 要和 list() 函数中的参数个数相同
  • 数组中的每个元素值会赋值 list() 函数中的每个参数, list() 将每个参数转为对应的变量
  • list() 只能接收索引数组
  • 转化时按索引的下标的顺序

预定义数组

概念

自动全局变量(又称超全局数组):

  • 包含了来自WEB服务器, 客户端, 运行环境和用户输入的数据
  • 这些数组比较特别, 全局范围内自动生效, 任意位置都可以直接使用这些数组
  • 用户不能自定义这些数组, 但这些数据操作方式又和我们自定义的数组操作方式一样
  • 在函数中直接可以使用这些数组
