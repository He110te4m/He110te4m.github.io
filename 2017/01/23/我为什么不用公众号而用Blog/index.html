我为什么不用公众号而用Blog

发表于 2017-01-23 | 分类于 杂谈

有位朋友问我，现在几乎没有人会输入网址打开网站了，基本都是靠 APP 推广和微信公众号，为什么我还选择建立 Blog 而不是微信公众号，又想到前几天忙着比赛也没更新博文，顺带更新下，所以有了本文~

本文所有内容仅代表作者本人观点，不参与任何争执与纠纷，最终解释权归作者(He110)所有

我拒绝使用公众号的原因主要有以下几个方面

  1. 我是一个极度崇尚自由的人，希望能在网络中有一片自己的天地，而微信公众号，会让我局限在微信所给予的框架中，无法逃离那种模式
  2. 我希望自己的文章能流传下去，能被搜索引擎记录，而微信中的文章保质期很短，基本是看完就扔在一边了，而且也只有搜狗搜索才能找到微信推文，文章的保质期和流传范围远远达不到我的需求
  3. 我希望自己的文章能够被不断修改、完善，而不是像微信推文一样发完就完了，虽说能再发，但是却很少会有人再看了，《Unix 编程艺术》中曾提到开源软件的定义：开源软件不是把源代码开出来就好了，而是要去和已有的其它软件互相融合，互相兼容，互相支持，我认为对于文章也是如此
  4. 我希望自己的文章能帮助更多的人，让更多的人看到，而不是像微信，在社交圈中传播一段时间就没有下文了，要知道在更多时候，文章本身的价值并没有文章的评论中的价值高，而大部分推文阅读者却看不到有价值的评论，并且评论的人也很难跟作者或其他用户进行深入的探讨，往往只是用户评论，作者回复，然后就没有下文了，很难逃离这种模式
  5. 我不想被流量绑架，对于公众号，尤其是个人订阅号，还是新开的来说，没有稳定的有质量的更新的话，我敢肯定关注人数大量流失是唯一的结局，也就是说，公众号其实是在绑架作者们，逼迫他们发布文章留住用户，但是，随着你写的推文的增多，以前的推文则会更加快速的被人遗忘
